The PRD (Product Requirement Document)

Project Name: GLANCE
Tagline: "Get smart in a glance."
Target: Grades 8-12 (Focus: JEE Physics & Math).
MVP Goal: A high-retention PWA that turns 1-minute of study into a daily habit.

Core User Journey:

    Trigger: Push Notification/WhatsApp at 7 PM (Timer counting down to 1 AM).

    Action: User solves 2 "Atomized" questions in < 60 seconds.

    Variable Reward: Instant XP, Knowledge Coins, and "Reflex Master" status.

    Investment: 1:00 AM Streak saved and "Error Bank" updated.
