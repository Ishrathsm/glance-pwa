Design System (Duo-Style)

Goal: Tactile, clean, and high-contrast.

    Colors:

        Background: #F9F9F7 (Notion Paper)

        Accent (Primary): #BEF264 (Cyber Yellow)

        Correct: #D7FFB8 (Background) | #58A700 (Text/Icon)

        Error: #FFDFE0 (Background) | #EA2B2B (Text/Icon)

        Ink: #1A1A1A (Charcoal)

    Typography: Nunito Sans

    Button Style:

        Border-Radius: 12px.

        3D Shadow: inset 0 -4px 0 0 rgba(0,0,0,0.15).

    The "Juice" (Animations):

        Haptics: 10ms buzz on selection, 40ms buzz on "Success Screen."

        Sound: "Ding" (Correct), "Thud" (Incorrect), "Ta-da!" (Goal Met).
