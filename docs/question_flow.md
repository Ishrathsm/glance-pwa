Screen 1: The Concept Phase

    Top: Progress Bar (3 segments).

    Hero: Context Box (Image/Main Text).

    Body: "Step 1: Concept" + 3 Options.

    Bottom: Check/Next Button (Disabled until selection).

Screen 2 & 3: The Unfolding

    Upon "Correct," the Concept question shrinks to a single line: "Concept: Volume is Constant ✅".

    The next question (Formula or Calculation) slides up from the bottom.

Screen 4: The Success Screen

    Hero: "That’s Correct!" headline.

    Body: Pro-Tip Box (Green background, max 20 words).

    Footer: Reward Summary (+15 XP, +5 Coins).

    Action: "Next Question" or "Finish Sprint."
